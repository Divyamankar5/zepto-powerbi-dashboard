# 📊 Zepto Sales & Operations Dashboard (Power BI)

## 🚀 Project Overview
This project presents an interactive Power BI dashboard analyzing Zepto's sales performance, order trends, and customer behavior. The dashboard provides business insights to support data-driven decision-making.

---

## 📌 Key Features
- Interactive dashboard with filters (Category, City, Date, Discount)
- KPI cards: Total Revenue, Total Orders, Avg Discount, Avg Price
- Monthly revenue trend analysis
- Discount impact on orders and revenue
- Product-wise and category-wise performance
- City-wise revenue analysis

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query (ETL)
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

---

## 📊 Key Insights
- Highest sales observed during mid-year (June–July)
- Snacks category generated the highest revenue
- Hyderabad and Bangalore are top-performing cities
- Higher discounts do not always result in higher revenue
- Certain products (Maggi, Parle-G) drive major sales

---

## 📂 Dataset
- Contains sales, orders, product, city, pricing, and discount details
- Data cleaned and transformed using Power Query
